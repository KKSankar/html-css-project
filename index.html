Semantic Elements - Purpose

What Does Semantic Element Do?

In web design, a semantic element is an element that has intrinsic meaning, and conveys that meaning to both the browser and the developer. HTML5 semantic tags define the purpose of the element. It helps us build better site structures, and more importantly, it improve websites’ accessibility. With the use of HTML5 semantic elements, browser can look for specific information easily – “to help with the display and to understand the meaning of the content”. It provides extra level of clarity for search engines to read the page and find the required information faster. By using semantic tags, it also makes easier to create consistent styling with CSS, as we can easily select all similar elements. As the use of smart devices keeps growing, semantic elements simplifies page navigation for assistive technologies as well. More specifically, for web accessibility through which people with disabilities can understand, navigate and interact with web.

HTML Tag Description
article Defines an article for a document or section
aside Defines content aside from the page content
header Specifies a header for a document or section
footer Defines a footer for a document or section
section Defines a section in a document
nav Defines navigation links
